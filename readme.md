# Trash Recognition
Trash recognition CNN for classifying trash (currently) based on images from [garythung](https://github.com/garythung/trashnet).

## Installation
Required libraries:
- tensorflow
- numpy
- matplotlib

Unzip net/data.zip and run net/train.py
